# Codex Fast Usage Examples

## 시작 프롬프트
```text
AGENTS.md와 docs/fast-track-context.md, docs/fast-track-plan.md, docs/fast-track-day-map.md를 읽고 앞으로 나를 GiftFlow fast-track study agent처럼 도와줘.
내가 Day N 시작이라고 하면 해당 단계에서 해야 할 구현, 공부 포인트, 테스트 포인트를 한 번에 너무 많이 말하지 말고 단계적으로 안내해줘.
```

## 진도 시작
```text
Day 7 시작
```

```text
2주 모드로 Day 10 시작
```

```text
오늘 뭐하지?
```

## 코드리뷰
```text
코드리뷰 해줘.
전체 재작성하지 말고 지금 가장 위험한 문제 3개만 짚어줘.
```

## 테스트 코드 코치
```text
지금 단계에서 꼭 필요한 테스트만 골라서 테스트 코드 같이 짜줘.
Given / When / Then 형식으로 설명해줘.
```

## CS 튜터
```text
CS 설명해줘.
오늘 단계와 연결되는 개념만 설명하고, 면접에서 어떻게 말할지도 알려줘.
```

## 너무 많을 때
```text
너무 많아. 오늘은 파일 1개만 수정하고 싶어.
다음 액션 1개만 줘.
```

## 구현 전에 설명만
```text
먼저 설명만 해줘. 내가 승인하면 코드로 가자.
```

## 공부 체크
```text
내가 이해했는지 확인 질문 2개만 내줘.
```

## 하루 마무리
```text
오늘 작업 기준으로 잘한 점 / 부족한 점 / 다음 단계 1개만 정리해줘.
```
